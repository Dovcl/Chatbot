# 챗봇 질문 가이드

## 📝 질문 방법

챗봇은 자연어로 질문하면 자동으로 데이터를 찾아서 답변해줍니다. 아래 예시를 참고하세요!

## 🎯 질문 예시

### 1. 날짜로 필터링하기

**지원하는 날짜 형식:**
- `2024-01-01` (하이픈)
- `2024/01/01` (슬래시)
- `2024.01.01` (점)

**질문 예시:**
```
2024-01-01 날짜의 데이터를 보여줘
2024/01/15의 모든 정보 알려줘
2024.01.20 데이터 찾아줘
```

### 2. 특정 컬럼 값으로 필터링하기

**형식:** `컬럼명 값`

**질문 예시:**
```
조사구간명 서울에서 데이터 보여줘
조사구간명 부산의 모든 정보 알려줘
조사구간명 서울에서 PH 값 말해줘
```

### 3. 날짜 + 컬럼 조합으로 필터링하기

**질문 예시:**
```
Date 2024-01-01, 조사구간명 서울에서 PH를 말해줘
2024-01-01 날짜에 조사구간명 서울의 PH 값 알려줘
2024/01/15, 조사구간명 부산에서 모든 데이터 보여줘
```

### 4. 특정 컬럼만 보기

질문에 컬럼명을 포함하면 해당 컬럼만 표시됩니다.

**질문 예시:**
```
PH 값만 보여줘
온도 데이터 알려줘
Date와 PH만 보여줘
```

### 5. 전체 데이터 보기

필터링 없이 모든 데이터를 보고 싶을 때:

**질문 예시:**
```
모든 데이터 보여줘
전체 데이터 알려줘
데이터 전체 보기
```

## 💡 실제 사용 예시

엑셀/CSV 파일에 다음과 같은 데이터가 있다고 가정:

| Date | 조사구간명 | PH | 온도 |
|------|-----------|----|----|
| 2024-01-01 | 서울 | 7.2 | 25.5 |
| 2024-01-01 | 부산 | 7.5 | 26.0 |
| 2024-01-02 | 서울 | 7.3 | 25.8 |

**질문과 답변 예시:**

1. **"2024-01-01 날짜의 데이터 보여줘"**
   → 2024-01-01 날짜의 서울과 부산 데이터 모두 표시

2. **"조사구간명 서울에서 PH를 말해줘"**
   → 서울의 PH 값들만 표시 (7.2, 7.3)

3. **"Date 2024-01-01, 조사구간명 서울에서 PH를 말해줘"**
   → 2024-01-01 날짜이면서 서울인 데이터의 PH 값만 표시 (7.2)

4. **"온도 데이터 알려줘"**
   → 모든 행의 온도 값만 표시

## ⚠️ 주의사항

1. **컬럼명 정확히 입력**: 엑셀/CSV 파일의 첫 번째 줄(헤더)에 있는 컬럼명을 정확히 사용하세요.
   - 예: 파일에 "조사구간명"으로 되어 있으면 "조사구간"이라고 하면 인식 안 될 수 있습니다.

2. **날짜 형식**: 날짜는 `YYYY-MM-DD`, `YYYY/MM/DD`, `YYYY.MM.DD` 형식만 인식합니다.

3. **대소문자 구분**: 컬럼명은 대소문자를 구분하지 않지만, 값은 구분할 수 있습니다.

4. **부분 일치**: 필터링할 때 값의 일부만 입력해도 찾을 수 있습니다.
   - 예: "조사구간명 서울" → "서울시", "서울특별시" 등도 찾을 수 있음

## 🔍 고급 사용법

### 여러 조건 조합
```
2024-01-01 날짜에 조사구간명 서울에서 PH와 온도를 알려줘
```

### 자연스러운 표현
```
서울 지역의 2024년 1월 1일 데이터에서 PH 값 좀 알려줘
부산에서 온도가 얼마야?
```

## ❓ 질문이 안 될 때

1. **"조건에 맞는 데이터를 찾을 수 없습니다"**
   - 날짜 형식이 맞는지 확인
   - 컬럼명이 정확한지 확인
   - 값이 정확히 일치하는지 확인

2. **"먼저 엑셀 또는 CSV 파일을 업로드해주세요"**
   - 파일을 먼저 업로드했는지 확인

3. **원하는 결과가 안 나올 때**
   - 브라우저 콘솔(F12)에서 오류 메시지 확인
   - 질문을 더 구체적으로 작성
   - 컬럼명을 정확히 입력
