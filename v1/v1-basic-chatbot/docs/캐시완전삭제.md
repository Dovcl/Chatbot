# 브라우저 캐시 완전 삭제 방법

## 🔥 즉시 해결 (가장 확실한 방법)

### Chrome/Edge
1. **F12**로 개발자 도구 열기
2. **Application** 탭 클릭
3. 왼쪽 사이드바에서 **Storage** 확장
4. **Clear site data** 버튼 클릭
5. 모든 항목 체크:
   - ✅ Local and session storage
   - ✅ IndexedDB
   - ✅ Web SQL
   - ✅ Cookies
   - ✅ Cache storage
6. **Clear site data** 클릭
7. 브라우저 완전 종료 후 재시작
8. `http://localhost:8000` 접속

### 또는 간단하게
1. 브라우저 주소창에 `chrome://settings/clearBrowserData` 입력 (Chrome)
2. 또는 `edge://settings/clearBrowserData` (Edge)
3. "캐시된 이미지 및 파일" 선택
4. "데이터 삭제" 클릭

## ✅ 확인 방법

브라우저 콘솔(F12)에서 확인:
- ✅ "✅ Supabase 초기화 완료" 메시지
- ❌ "Firebase 초기화 완료" 메시지가 없어야 함
- ❌ "Firebase 설정을 확인해주세요" 메시지가 없어야 함

## 🚨 여전히 문제가 있으면

1. **다른 브라우저에서 테스트** (Chrome → Firefox)
2. **시크릿 모드**에서 테스트 (`Cmd + Shift + N`)
3. **서버 재시작** 확인
