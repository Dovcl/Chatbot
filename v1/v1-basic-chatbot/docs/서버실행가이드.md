# 로컬 서버 실행 가이드

## ⚠️ 중요: 파일을 직접 열지 마세요!

HTML 파일을 더블클릭하거나 `file://` 프로토콜로 열면 CORS 오류가 발생합니다.
반드시 **로컬 서버**를 통해 실행해야 합니다.

## 서버 실행 방법

### 방법 1: Python 사용 (권장)

터미널에서 프로젝트 폴더로 이동 후:

```bash
# Python 3
python3 -m http.server 8000

# 또는 Python 2
python -m http.server 8000
```

### 방법 2: Node.js 사용

```bash
# http-server 설치 (처음 한 번만)
npm install -g http-server

# 서버 실행
http-server -p 8000
```

### 방법 3: VS Code Live Server

1. VS Code에서 프로젝트 폴더 열기
2. `index.html` 파일 우클릭
3. "Open with Live Server" 선택

## 접속 방법

서버 실행 후 브라우저에서:
```
http://localhost:8000
```

## 서버 중지

터미널에서 `Ctrl + C` 누르기

## 확인 사항

브라우저 주소창이 다음처럼 보여야 합니다:
- ✅ `http://localhost:8000` (정상)
- ❌ `file:///Users/...` (잘못됨 - 서버 사용 필요)

## 문제 해결

### "포트가 이미 사용 중" 오류
```bash
# 다른 포트 사용
python3 -m http.server 8001
# 또는
python3 -m http.server 3000
```

### "python3 명령을 찾을 수 없음"
- Python이 설치되어 있는지 확인
- 또는 Node.js의 http-server 사용
